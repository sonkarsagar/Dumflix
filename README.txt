# Dumflix - Netflix Clone

Dumflix is a Netflix-style web application built using React and Redux for learning and practice purposes. It includes Firebase authentication, TMDB API integration for real movie data, and a GPT-based movie suggestion feature.

---

## Features
- User authentication (Login/Logout) with Firebase
- Movie data fetched from TMDB API
- GPT-powered movie search suggestions
- State management using Redux Toolkit
- Netflix-like user interface
- Responsive design using Tailwind CSS

---

## Tech Stack
- React.js
- Redux Toolkit
- Firebase Authentication
- TMDB API
- Tailwind CSS

---

## Setup Instructions
1. Clone the repository  
   `git clone https://github.com/sonkarsagar/Dumflix.git`

2. Install dependencies  
   `npm install`

3. Add your TMDB and Firebase API keys to `.env`

4. Start the application  
   `npm start`
